# link-shortener
Simple link-shortener built with Java Spring Boot to show backend and API development skills.


## Features
- Input a URL and receive a shortened version
- Short URLs are generated and stored in-memory (no database required)
- REST API built with Spring Boot
- Designed for portfolio / lab demonstration


## Techstack
- Java 21
- Spring Boot 4.0.3
- Maven


## Getting Started

###

### Installation

# Clone the repo
git clone git@github.com:your-username/link-shortener-lab.git

# Build the project
mvn clean install

# Run
mvn spring-boot:run
